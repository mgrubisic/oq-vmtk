Intensity Measure Processing
============================

Demonstrates how to compute intensity measures (IMs) from ground-motion records using the ``imcalculator`` class. Covers response spectra, peak ground motion parameters (PGA, PGV, PGD), average spectral acceleration, Arias Intensity, CAV, significant duration, and FIV3.

`Open notebook on GitHub <https://github.com/GEMScienceTools/oq-vmtk/blob/incremental-dynamic-analysis/demos/IntensityMeasureProcessing/IntensityMeasureProcessing.ipynb>`_
