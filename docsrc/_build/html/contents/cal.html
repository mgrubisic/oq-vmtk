5. Model Calibration

The calibration class transforms Single-Degree-of-Freedom (SDOF) spectral capacity parameters into Multi-Degree-of-Freedom (MDOF) storey force-deformation relationships. It handles building classification, mass and stiffness matrix assembly, eigenvalue-based modal analysis, period scaling, storey force-drift distribution, and optional OpenSees static pushover verification.
