11.7. Modified Cloud Analysis

Demonstrates how to postprocess the results of a Modified Cloud Analysis (MCA) using the postprocessor class. Covers lognormal fragility fitting with bootstrap and classical Bayesian MCMC estimation (Jalayer et al., 2017), GLM methods, and vulnerability function derivation.
